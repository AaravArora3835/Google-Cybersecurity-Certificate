## Curriculum Summary

### Course 1: Foundations of Cybersecurity
Introduction to cybersecurity roles, threat landscapes, security frameworks (NIST CSF, CIA Triad), and industry career pathways.

### Course 2: Play It Safe - Manage Security Risks
Risk assessment methodologies, security audits, threat modeling, compliance frameworks (GDPR, HIPAA), and security governance principles.

### Course 3: Connect and Protect - Networks and Network Security
Network architecture, TCP/IP protocols, network hardening techniques, firewall configuration, VPN implementation, and network security monitoring.

### Course 4: Tools of the Trade - Linux and SQL
Linux command line fundamentals, file system navigation, permissions management, bash scripting basics, SQL queries, database filtering, and joins.

### Course 5: Assets, Threats, and Vulnerabilities
Asset classification, vulnerability assessment, CVE and CVSS frameworks, patch management strategies, threat intelligence, and attack surface analysis.

### Course 6: Sound the Alarm - Detection and Response
Incident response lifecycle, SIEM platform operations, log analysis and correlation, IDS/IPS configuration, digital forensics fundamentals, and SOC workflows.

### Course 7: Automate Cybersecurity Tasks with Python
Python programming fundamentals, string and list manipulation, file operations, regular expressions, security automation scripting, and log parsing.

### Course 8: Put It to Work - Prepare for Cybersecurity Jobs
Resume development, technical interview preparation, portfolio building, STAR method communication, and professional networking strategies.

### Course 9: Accelerate Your Job Search with AI
AI-powered job search tools, resume optimization platforms, interview simulation systems, and ethical AI integration in career development.
